# Product Vision

This project exists to let two long-distance partners send small physical notes to each other.

The experience should feel warm, simple, and personal:
- Log in
- Write a short note
- Send it
- The other person’s printer eventually prints the note as a physical receipt-style message

This is not meant to be a general-purpose messaging app. It is intentionally small, intimate, and private.

## Users

There are two intended users:
- Me
- My partner

Each user should have their own secure login.

## Core user experience

A user should be able to:
- Log in securely
- Write a message
- Choose/send to the other person
- See whether the message is pending, printed, or failed
- View recent sent/received notes

## Tone

The UI should feel modern, cozy, and low-friction. Avoid enterprise/admin-style UI.
